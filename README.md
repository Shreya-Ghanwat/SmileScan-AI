
# 🦷SmileScan-AI

SmileScan-AI is a deep learning-based dental disease classification system that analyzes oral images and dental X-rays using CNN models.

---

## Features

- Dual-model system (Oral + X-ray)
- Disease classification with confidence scores
- Explainable AI using Grad-CAM
- Streamlit web interface
- Report generation

---

## Models

- ResNet50 → Oral image classification
- ResNet50 → X-ray analysis

---

## Tech Stack

Python, PyTorch, Streamlit, NumPy, PIL

---

## Setup

git clone https://github.com/Shreya-Ghanwat/SmileScan-AI.git  
cd SmileScan-AI  

python -m venv venv  
venv\Scripts\activate  

pip install -r requirements.txt
